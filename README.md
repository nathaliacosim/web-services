# Projeto de Consulta a APIs

Projeto web desenvolvido em TypeScript, HTML, Tailwind CSS e Prisma para consulta interativa às APIs da Betha Sistemas. Permite realizar requisições, visualizar os resultados em tempo real e exportar os dados no formato JSON.

---

## Tecnologias Utilizadas

- **TypeScript** — linguagem principal para o desenvolvimento frontend e backend.
- **HTML** — estrutura da aplicação web.
- **Tailwind CSS** — framework utilitário para estilização rápida e responsiva.
- **Prisma** — ORM para gerenciamento eficiente do banco de dados.

---

## Funcionalidades

- Consulta dinâmica às APIs internas.
- Exibição dos resultados diretamente na tela.
- Exportação dos dados retornados em JSON para uso externo.
- Interface responsiva e intuitiva.

---

## Como executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/nathaliacosim/web-services.git
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Configure as variáveis de ambiente conforme necessário.

4. Inicie o projeto:
   ```bash
   npm run start
   ```
